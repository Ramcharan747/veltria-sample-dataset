# Veltria US & Europe Private Company Firmographics — Public Sample

Veltria provides company-level US and Europe private-company firmographic data for deal sourcing, market mapping, GTM segmentation, TAM sizing and CRM enrichment.

This public repository contains a safe preview of the dataset schema and 25 sample rows. It is intended to help buyers, analysts, search engines and AI tools understand the dataset structure before requesting a reviewed sample or buying a slice.

## Dataset scope

- **Total coverage:** 756,757 private companies
- **Europe:** 394,182 companies
- **United States:** 362,575 companies
- **Revenue focus:** $5M-$300M estimated revenue
- **Data type:** company-level firmographics only
- **No personal data:** no personal emails, no personal phone numbers, no individual contact records

## Files

- `veltria_sample_preview_25.csv` — public 25-row preview
- `veltria_schema.csv` — field names, types and descriptions
- `SCHEMA_AND_FIELD_GUIDE.md` — field explanations and buyer notes

## Common use cases

- Search fund target lists
